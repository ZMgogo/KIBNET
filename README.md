# KIBNET
中国科学院昆明植物研究所（KIB）断网重连软件

运行原理：
  每隔5秒向百度发送 ping 包，百分百丢失后会启动断网重连功能。

断网重连原理：
  使用 JavaScript 加密用户信息后提交联网请求。除了加密部分外全都由 Python 实现。
  
效率：正常情况下10秒内能实现重连网络。
